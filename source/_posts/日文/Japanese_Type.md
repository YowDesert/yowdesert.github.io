# 日文筆記｜日文的各種「型」（活用）完整整理

> 本筆記整理日文動詞的各種「活用型」，適用於 N5～N4  
> 核心觀念：**先判斷動詞種類 → 再決定怎麼變形**

---

## 一、日文動詞的三大類

### Ⅰ類動詞（五段動詞）
- 結尾多為：う・つ・る・む・ぶ・ぬ・く・ぐ・す
- 例：
  - 書く、読む、話す、待つ、遊ぶ、死ぬ

### Ⅱ類動詞（一段動詞）
- 結尾是「る」
- 前一個音為 **い / え 音**
- 例：
  - 食べる、見る、起きる、教える

> 常見例外（明明是る＋い/え，卻是五段）
> - 走る
> - 入る
> - 知る
> - 切る
> - 帰る
> - 要る

### Ⅲ類動詞（不規則）
- する
- くる

---

## 二、動詞的六大基本活用型

---

## 1️⃣ 未然形（ない形的基礎）

### 功能
- 表示否定、尚未發生
- 常接：
  - ～ない
  - ～れる / ～られる（被動）
  - ～せる / ～させる（使役）

### 變化規則
| 動詞類型 | 變化方式 |
|--------|----------|
| 五段動詞 | 尾音變成「あ段」 |
| 一段動詞 | 去る |
| 不規則 | する→し／くる→こ |

> 例外 : 「う」結尾的動詞例外，要變成「わ」
### 例子
- 書く → 書かない  
- 読む → 読まない  
- 食べる → 食べない  
- する → しない  
- くる → こない  

---

## 2️⃣ 連用形（ます形的基礎）

### 功能
- 接續動作、禮貌語
- 常接：
  - ～ます
  - ～たい
  - ～た
  - ～て
  - ～ながら

### 變化規則
| 動詞類型 | 變化方式 |
|--------|----------|
| 五段動詞 | 尾音變成「い段」 |
| 一段動詞 | 去る |
| 不規則 | する→し／くる→き |

### 例子
- 書く → 書きます  
- 読む → 読みたい  
- 食べる → 食べます  
- する → します  
- くる → きます  

---

## 3️⃣ 終止形（字典形）

### 功能
- 句子結尾（普通形）
- 辭典查詢用

### 例子
- 私は日本語を**勉強する**。
- 明日映画を**見る**。

---

## 4️⃣ 連體形（修飾名詞）

### 功能
- 修飾名詞（～的～）
- 現代日文中，**形狀與終止形相同**

### 例子
- 私が**読む**本  
- 日本語を**勉強する**人  
- 昨日**買った**かばん  

---

## 5️⃣ 假定形（ば形）

### 功能
- 表示「如果～就～」
- 常見句型：
  - ～ばいい
  - ～ばわかる
  - ～ばよかった

### 變化規則
| 動詞類型 | 變化方式 |
|--------|----------|
| 五段動詞 | 尾音變成「え段＋ば」 |
| 一段動詞 | 去る＋れば |
| 不規則 | する→すれば／くる→くれば |

### 例子
- 書く → 書けば  
- 読む → 読めば  
- 食べる → 食べれば  
- する → すれば  
- くる → くれば  

---

## 6️⃣ 命令形

### 功能
- 命令、強烈請求
- 常見於：
  - 男性用語
  - 標語、漫畫、口號

### 變化規則
| 動詞類型 | 變化方式 |
|--------|----------|
| 五段動詞 | 尾音變成「え段」 |
| 一段動詞 | 去る＋ろ（口語）／よ（書面） |
| 不規則 | する→しろ／くる→こい |

### 例子
- 書く → 書け  
- 待つ → 待て  
- 食べる → 食べろ  
- する → しろ  
- くる → こい  

---

## 三、超重要衍生型

---
## 補充 1：て形、た形一定要記「音便規則」

### 五段動詞て形規則 : 

| 結尾    | て形 |
| ----- | -- |
| う・つ・る | って |
| む・ぶ・ぬ | んで |
| く     | いて |
| ぐ     | いで |
| す     | して |

> 例外：行く → 行って（不是 行いて）

---

## た形規則

| て形 | た形 |
| -- | -- |
| って | った |
| んで | んだ |
| いて | いた |
| いで | いだ |
| して | した |

---

## 7️⃣ て形

### 功能
- 連接動作
- 請求（～てください）
- 正在進行（～ている）
- 允許 / 禁止（～てもいい／～てはいけない）

### 例子
- 本を**読んで**ください。
- 日本語を**勉強しています**。

---

## 8️⃣ た形

### 功能
- 過去
- 完成
- 經驗（～たことがある）

### 例子
- 昨日映画を**見た**。
- 日本へ**行ったことがある**。

---

## 9️⃣ ない形

### 功能
- 否定
- 常見句型：
  - ～ないでください
  - ～なくてもいい

### 例子
- 行かない。
- 食べなくてもいい。


---
## 🔟 可能形（能夠～）

### 功能
- 表示能力、可能性（能做某事）

### 變化規則
| 動詞類型 | 變化方式 |
|--------|----------|
| 五段動詞 | 尾音變成「え段＋る」 |
| 一段動詞 | 去る＋られる |
| 不規則 | する→できる／くる→こられる |

### 例子
- 書く → 書ける（能寫）
- 読む → 読める（能讀）
- 食べる → 食べられる（能吃）
- する → できる
- くる → こられる

> 口語常見 : ら抜き言葉（省略「ら」）
> 食べられる → 食べれる（口語）

---

## 四、快速記憶整理（考試用）

- **ます／て／た／たい → 連用形**
- **ない／被動／使役 → 未然形**
- **如果～ → 假定形（ば）**
- **命令 → 命令形**
- **修飾名詞 → 連體形**

---

## 五、學習建議
1. 先熟 **ます形・ない形・て形**
2. 再補 **ば形、命令形**
3. 每個動詞都問自己一句話：  
   👉「我要接什麼？那我要用哪一型？」
